Playing with tags


Let's play with tags in HTML

Tags are a necessary part of HTML and mastering them is key to a successful website.

Here I will list just some of the tags and what they do:

Some HTML Tags

html
HTML is a basic tag that tells the browser what is to follow and to open the file in the browser.
Headers
h1, h2, h3, h4, h5, h6 are all Header tags that vary in size from largest to smallest.
p tags
p tags tell the browser this is a paragraph.
